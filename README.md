# FCApy Tutorials
A repository to contain tutorials for [FCApy](https://github.com/EgorDudyrev/FCApy) package.

The repository is organised as follows:
* [ClassTutorials](https://github.com/EgorDudyrev/FCApy_tutorials/tree/main/ClassTutorials)
directory contains demonstrations of basic usage of FCApy classes like:
  * [Formal Context](https://github.com/EgorDudyrev/FCApy_tutorials/blob/main/ClassTutorials/Formal%20Context.ipynb)
  * [Concept Lattice](https://github.com/EgorDudyrev/FCApy_tutorials/blob/main/ClassTutorials/ConceptLattice.ipynb)
  * [MVContext](https://github.com/EgorDudyrev/FCApy_tutorials/blob/main/ClassTutorials/MVContext.ipynb)
* [Lattice Visualization](https://github.com/EgorDudyrev/FCApy_tutorials/tree/main/Lattice%20Visualization)
directory demonstrates a case study to create a colourful lattice visualization via FCApy
 
More tutorials to come when FCApy will become more profound.
